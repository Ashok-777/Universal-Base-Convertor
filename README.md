# Universal Base Converter

A powerful, client-side **Universal Base Converter** that allows conversion between **Binary, Octal, Decimal, and Hexadecimal** numbers. It supports multiple modes including **Unsigned Integer, Signed Integer (Two's Complement), Fixed-Point, and Floating-Point (IEEE 754)** with detailed step-by-step explanations.

This tool is perfect for programmers, students, and engineers who need a fast and interactive way to understand number base conversions.

## üß™ How It Works

1. Select the **conversion mode** (Unsigned, Signed, Fixed-Point, Floating-Point).  
2. Select the **source base** (Binary, Octal, Decimal, Hexadecimal).  
3. Enter the **number** in the input field.  
4. The tool **automatically updates results** in all target bases.  
5. Click **Copy** to copy the converted value.  
6. Click **Expand Explanation** to see detailed step-by-step conversion logic.  

> The signed integer mode allows you to specify **bit width** (8, 16, 32, 64 bits) for two's complement conversions.
